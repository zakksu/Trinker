# Drop Hera .aoe2record files here

TRINKER scans this folder when you run **SETUP_HERA_COACH.bat** or **Settings → Build Hera Coach**.

Sources for Hera replays:
- [AoE2 Insights](https://aoe2insights.com/) — pro player replays
- Tournament VODs saved as `.aoe2record` from spectating
- Your own spectated ranked games where Hera is in the lobby

Team games and 1v1 both work — TRINKER finds any match where **Hera** appears as a player name.
